⚡ Typing Speed Tester

An interactive and responsive Typing Speed Tester Web Application developed using HTML, CSS, and JavaScript.

🚀 Live Demo

GitHub Pages link here after deployment:

https://kinetickishore2005-creator.github.io/Typing-speed-test-GK/

📌 Features

- ⏱ Real-time Typing Speed (WPM) Calculation
- 🎯 Accuracy Tracking
- ❌ Mistake Counter
- 🌙 Dark / Light Mode Toggle
- 🔄 Random Quote Generator
- 📊 Typing Progress Bar
- 📱 Fully Responsive Design
- ✨ Modern Glass morphism UI
- 🔁 Restart Test Functionality
- ⚡ Interactive Animations and Hover Effects

🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages

▶ How to Run the Project

1. Download or clone the repository
2. Open the project folder in VS Code
3. Install Live Server extension
4. Open `index.html`
5. Right click → Open with Live Server
